# 第三届人工智能与智算发展论坛

一个基于 Next.js 的抽奖系统，包含抽奖大屏和独立后台。

## 功能

- 滚动号码牌抽奖
- 多轮次配置
- 每轮中奖数量配置
- 号码池数量自动生成
- 预设中奖号码
- 已中奖号码不重复中奖
- 抽奖前号码牌显示问号
- 抽奖记录导出
- 深紫科技风抽奖大屏

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000` 查看抽奖大屏。

后台入口在抽奖大屏左上角 `VIBE CODING`，也可以直接访问：

```text
http://localhost:3000/admin
```

## 构建

```bash
npm run build
```
